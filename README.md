# Array Duel

中文名：斗阵

Array Duel 是一款手机优先的五行阵法卡牌对战原型。玩家围绕木、火、土、金、水与中台六个阵位布阵，通过灵力相减、五行生克、里侧防守、相生联协和阵眼效果击败对手。

> 资源生成约定：使用 image2 生成 UI、背景、头像、卡背等界面资产时，统一通过 `scripts/generate_ui_art.py` 增加 target 并执行，避免散落的临时调用。

## 核心玩法

- 双方各有 6 个阵位：木、火、土、金、水、中台。
- 卡牌分为攻击、防守、恢复、反击、销毁、永续、阵眼。
- 五行槽按阵位放置；中台可以放任意属性卡，阵眼只能放在中台。
- 攻击卡最多 3 张；防守、反击、销毁最多 3 张；恢复、永续和阵眼不占攻防名额。
- 中台始终表侧展示。
- 非中台卡可以表侧或里侧放置，里侧卡隐藏具体效果。
- 攻击卡使用灵力相减结算，克制或借生关系会让灵力翻倍。
- 防守回合可以触发防守卡，并沿五行相生方向询问是否联协；反击卡会像陷阱卡一样读取反击属性、被攻击槽位和攻击来源的五行生克来连锁补阵、补手、削弱、重铸或蓄势。
- 攻击位对撞会结算双向溢出，撞不过对方攻击卡时会受到反震伤害。
- 每回合获得 1 枚灵石，可补灵，也可消耗 2 枚灵石炼石抽牌或从灵宝碎片重铸卡牌。
- 主阶段可以用手卡替换任意合法卡槽。
- 主阶段可以把手牌作为多模式法诀打出，按目标、手牌代价和灵宝碎片限制发动。
- 被破坏、替换、联协消耗和结束阶段弃置的卡会化为灵宝碎片；结束阶段手卡超过 6 张时，需要弃到 6 张。

## 补充包：五行连锁

本版新增 50 张补充包卡牌，每个五行 10 张，核心方向是把亏卡行为转化为可循环资源：

- 木系偏定向检索和木生火启动，能把灵宝碎片炼成相生属性手牌。
- 火系偏焚化手牌爆发、换抽和灼阵，能用手牌成本换进攻节奏。
- 土系偏炼化灵宝碎片补灵、产灵石和护阵，让资源沉淀变成场面厚度。
- 金系偏削灵、拆阵、焚手夺灵石，并通过金生水补水系续航。
- 水系偏从灵宝碎片重铸、滤抽和续航，把防守资源重新变成手牌。
- 反击卡按五行生克连锁，不再固定按属性分工；攻击空槽或攻击攻击位都可能触发补阵、无效、重铸、抽牌和临时蓄势。

## 回合阶段

1. 准备阶段：获得灵石，触发永续效果，随机抽 1 张手卡。
2. 主1：放置或替换卡牌，执行一次灵石动作。
3. 发动阶段：主动发动 1 个卡槽或手卡效果。
4. 主2：继续放置或替换卡牌，若未使用灵石动作可继续处理。
5. 结束阶段：处理手卡上限，然后交换回合。

## 本地运行

不需要构建步骤。基础玩法可静态运行；若启用默认大模型对手，线上服务器需要支持 `web/ai-proxy.php` 并配置私有 API Key。

```bash
cd web
python3 -m http.server 5173
```

然后打开：

```text
http://localhost:5173/
```

## 部署

把 `web/` 目录作为站点根目录发布即可。默认大模型对手通过 `ai-proxy.php` 访问服务器私有 Key，配置方式见 [web/DEPLOY.md](web/DEPLOY.md)。

可选平台：

- GitHub Pages
- Vercel
- Netlify
- Nginx 静态目录

手机访问后，可以通过浏览器菜单添加到主屏幕，作为 PWA 使用。

## 美术与音频来源

- 2D 像素法术特效来自 OpenGameArt 的 [Pixel Art Spells](https://opengameart.org/content/pixel-art-spells)，作者 DevWizard，授权 CC0。项目内使用了火球、火焰爆发、藤蔓、岩石、水流、光刃、护盾、法术火花等 PNG spritesheet。
- 大爆炸特效来自 OpenGameArt 的 [Explosion](https://opengameart.org/content/explosion-7)，作者 BenHickling，授权 CC0，用于销毁、周天和高爆发反馈。
- 斩击特效来自 OpenGameArt 的 [Weapon Slash - Effect](https://opengameart.org/content/weapon-slash-effect)，作者 Cethiel，授权 CC0，用于金系、火系、水系攻击表现。
- 冲击波特效来自 OpenGameArt 的 [New Moon Shock Waves Sprite](https://opengameart.org/content/new-moon-shock-waves-sprite)，作者 bongbonggun，授权 CC0，用于克制、联协和破阵冲击。
- 背景音乐文件位于 `web/assets/xianxia-duel-aigei.mp3`，请根据实际下载来源保留相应授权说明。
