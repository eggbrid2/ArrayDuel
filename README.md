# Array Duel

中文名：斗阵

Array Duel 是一款手机优先的五行阵法卡牌对战原型。玩家围绕木、火、土、金、水与中台六个阵位布阵，通过灵力相减、五行生克、里侧防守、相生联协和阵眼效果击败对手。

当前线上试玩地址：

<http://152.136.18.122/wuxing/>

## 核心玩法

- 双方各有 6 个阵位：木、火、土、金、水、中台。
- 卡牌分为攻击、防守、恢复、反击、销毁、永续、阵眼。
- 五行槽按阵位放置；中台可以放任意属性卡，阵眼只能放在中台。
- 中台始终表侧展示。
- 非中台卡可以表侧或里侧放置，里侧卡隐藏具体效果。
- 攻击卡使用灵力相减结算，克制或借生关系会让灵力翻倍。
- 防守回合可以触发防守卡，并沿五行相生方向询问是否联协。
- 每回合获得 1 枚灵石，可为卡槽补充灵力。
- 主阶段可以用手卡替换任意合法卡槽。
- 结束阶段手卡超过 6 张时，需要弃到 6 张。

## 回合阶段

1. 准备阶段：获得灵石，触发永续效果，随机抽 1 张手卡。
2. 主1：放置或替换卡牌，补灵。
3. 发动阶段：主动发动 1 个卡槽或手卡效果。
4. 主2：继续放置或替换卡牌，若未补灵可补灵。
5. 结束阶段：处理手卡上限，然后交换回合。

## 本地运行

不需要构建步骤。基础玩法可静态运行；若启用默认大模型对手，线上服务器需要支持 `web/ai-proxy.php` 并配置私有 API Key。

```bash
cd web
python3 -m http.server 5173
```

然后打开：

```text
http://localhost:5173/
```

## 部署

把 `web/` 目录作为站点根目录发布即可。默认大模型对手通过 `ai-proxy.php` 访问服务器私有 Key，配置方式见 [web/DEPLOY.md](web/DEPLOY.md)。

可选平台：

- GitHub Pages
- Vercel
- Netlify
- Nginx 静态目录

手机访问后，可以通过浏览器菜单添加到主屏幕，作为 PWA 使用。
