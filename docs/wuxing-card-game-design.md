# 五行阵斗 0.1 版设计文档

## 1. 游戏定位

《五行阵斗》是一款双人对战卡牌游戏。玩家不是使用传统手牌连续出牌，而是在自己的六个阵位上经营一座五行阵。每回合主动只能使用一个卡槽，但可以通过永续卡、里侧防守、补灵和五行相生联协，在对手回合形成反击。

核心体验：

- 六个卡槽构成阵法，而不是传统手牌区。
- 攻击、防御、治疗、销毁都是一次性卡，发动后离场。
- 永续卡长期占槽，有灵力和持续收益，但会被同属性或五行关系针对。
- 主动回合只动一个槽，选择很重。
- 被动回合可触发防御、永续、联协和额外灵力增强。
- 抽卡按五行牌组进行，玩家经营的是五个属性的小牌库。

## 2. 场地

每名玩家有 6 个卡槽：

```text
[ 木 ] [ 火 ] [ 土 ]
[ 水 ] [ 中 ] [ 金 ]
```

五行槽：

- 木槽
- 火槽
- 土槽
- 金槽
- 水槽

中台：

- 可放任意属性卡。
- 不属于任意五行槽。
- 作为阵眼存在。只要中台有卡，对手不能压阵你的空五行槽。
- 可以放置阵眼卡，形成持续生效的中台核心。
- 普通一次性卡放在中台发动时，会获得中台发动规则。

## 3. 卡牌基础

每张卡有：

- 属性：木、火、土、金、水
- 类型：攻击、防御、治疗、销毁、永续、阵眼
- 灵力：卡牌的统一基础强度。攻击、防御、治疗、反击、销毁、永续承受消耗都使用灵力结算。
- 效果：主动、被动、联协、补灵

一次性卡：

- 攻击卡、防御卡、治疗卡、销毁卡都是一次性卡。
- 发动后进入弃牌区。
- 所在卡槽变空。

永续卡：

- 持续占用卡槽。
- 拥有灵力。
- 被攻击或销毁时消耗灵力。
- 灵力降为 0 时破坏，进入弃牌区。
- 可被同属性、克制属性、或“生攻击方属性”的攻击卡攻击。

阵眼卡：

- 只能放置在中台。
- 拥有五行属性。
- 持续生效，不会因回合结束离场。
- 没有灵力，不能被普通攻击破坏。
- 只能被销毁卡、指定“破坏阵眼”的效果、或玩家主动更换移除。
- 玩家主动更换阵眼时，旧阵眼进入弃牌区。
- 阵眼卡占用中台，因此中台不能同时放置其他卡。

公开信息：

- 除阵眼外，卡牌放置时可以选择表侧或里侧。
- 攻击卡里侧放置时仍占攻击位；防御、治疗、反击、销毁卡里侧放置时仍占防守位；永续卡里侧放置时仍占永续位。
- 里侧卡的属性、所在卡槽、攻击/防守/永续姿态公开，具体卡名和效果隐藏。
- 表侧卡的卡名、类型、灵力和效果公开。
- 阵眼卡始终正面公开。
- 里侧卡上的额外灵力公开。
- 若效果需要检查里侧卡属性，五行槽使用所在卡槽属性；中台里侧卡使用卡牌自身属性。

## 4. 初始设置

建议第一轮测试使用：

- 玩家生命：24
- 每个属性牌组：20 张
- 总卡池：100 张基础五行卡，另有 10 张阵眼扩展卡。
- 每名玩家可以使用同一套 100 张测试卡，也可以镜像对战。
- 起始填阵：木、火、土、金、水五槽分别从对应属性牌组抽 1 张。中台从任意一个属性牌组抽 1 张。
- 起始手卡：每名玩家额外抽 5 张手卡。
- 每回合获得灵石：1 枚。
- 每个卡槽最多通过灵石补充 2 点额外灵力。
- 每回合主动行动：只能使用 1 个卡槽效果。

## 5. 五行关系

相生：

```text
木生火
火生土
土生金
金生水
水生木
```

相克：

```text
金克木
木克土
土克水
水克火
火克金
```

## 6. 攻击合法目标

攻击卡可以攻击三类目标：

1. 同属性目标。
2. 被攻击卡属性克制的目标。
3. 生出攻击卡属性的目标。

例如火属性攻击卡可以攻击：

- 火：同属性。
- 金：火克金。
- 木：木生火。

完整表：

| 攻击属性 | 可攻击目标 |
|---|---|
| 木 | 木、土、水 |
| 火 | 火、金、木 |
| 土 | 土、水、火 |
| 金 | 金、木、土 |
| 水 | 水、火、金 |

中台攻击：

- 中台卡使用卡牌自身属性判断能否被攻击。
- 若中台为空，不能被压阵。
- 中台为空时，只会解除对五行空槽的压阵保护。
- 中台上的攻击卡主动发动时，可以攻击对方任意五行卡槽。
- 若中台攻击的目标符合该攻击卡的同属性、相克、或生我关系，正常获得对应收益。
- 若中台攻击的目标不符合五行关系，只使用基础灵力相减，不能获得五行倍率，也不能触发压阵。
- 中台攻击不能直接攻击对方玩家，除非卡牌效果特别写明。

攻击结算：

- 攻击不再使用独立伤害值，统一使用灵力相减。
- 攻击方灵力 = 攻击卡基础灵力 + 额外灵力。
- 若攻击方属性克制目标属性，或目标属性生出攻击方属性，攻击方灵力 x2。
- 同属性时双方都不翻倍。
- 攻击攻击位卡时，防守方也用自身灵力参与相减；若防守方属性克制攻击方属性，或攻击方属性生出防守方属性，防守方灵力 x2。
- 攻击位无论表侧或里侧，被攻击时都不发动效果，只翻开并进行灵力对比。
- 攻击方最终灵力大于防守方最终灵力时，防守方攻击位卡被破坏。
- 攻击方最终灵力小于或等于防守方最终灵力时，防守方攻击位卡不被破坏。
- 防守方灵力溢出不会反伤攻击方或攻击玩家。

攻击永续卡、防守卡或中台普通卡时：

- 只计算攻击方最终灵力，然后从目标当前灵力中扣除。
- 目标当前灵力扣到 0 或以下时，该卡被破坏。
- 若目标剩余灵力大于 0，该卡继续留场。

示例：

- 火打火：1 - 1，防守方不被破坏。
- 火打木：1 x2 - 1，火借木生火，若结果更高则木攻击位被破坏。
- 火打水：1 - 1 x2，水克火，水攻击位不被破坏，溢出不反伤。

## 7. 压阵

当攻击卡选择一个合法五行目标槽时：

- 如果该槽有卡，正常攻击或结算效果。
- 如果该槽为空，且防守方中台也为空，攻击卡可以进入该空槽成为压阵卡。
- 压阵卡不能攻击，不能触发原本卡牌效果。
- 压阵卡阻止该槽补阵。
- 压阵卡可被同属性或克制属性销毁卡清除。

阵势崩溃胜利：

- 若一名玩家的五个五行槽中有 3 个被对手压阵，该玩家立即失败。

## 8. 回合流程

### 8.1 主动回合

1. 准备阶段
   - 获得 1 枚灵石。
   - 触发己方准备阶段永续效果。
   - 每回合随机抽 1 张卡加入手卡。
   - 不再自动补卡槽。
   - 空卡槽会持续保持为空，直到玩家主动从手卡放置卡牌。
   - 若抽到攻击卡但己方已有 3 张攻击位卡，该卡仍加入手卡，不能直接进入卡槽。

2. 主1
   - 可以将手卡放入对应卡槽，也可以替换该槽已有卡。
   - 放置时选择表侧或里侧；阵眼只能表侧放置。
   - 被替换的旧卡进入弃牌区。
   - 可以支付 1 枚灵石，为己方任意一个卡槽补充 1 点额外灵力。
   - 每槽最多 2 点额外灵力。
   - 主1不能发动卡牌效果。

3. 发动阶段
   - 只能主动使用 1 个卡槽效果。
   - 可发动一次性卡、永续主动效果或阵眼效果。
   - 发动卡牌后不会自动结束回合。

4. 主2
   - 可以继续将手卡放入对应卡槽，或替换该槽已有卡。
   - 放置时同样可以选择表侧或里侧；阵眼只能表侧放置。
   - 如果本回合还没有补灵，可以在主2补灵。
   - 手卡放置规则：五行卡只能放入或替换对应五行槽，阵眼卡只能放入或替换中台。
   - 若手卡是攻击卡且己方已有 3 张攻击位卡，不能放置。

5. 结束阶段
   - 清理本回合临时效果。
   - 检查生命归零和阵势崩溃。
   - 玩家点击结束回合后，回合才会交给对手。

### 8.2 被动回合

对方回合中，己方可以触发：

- 被攻击槽的防御卡。
- 永续卡被动效果。
- 五行相生联协。
- 额外灵力增强。

被动触发不消耗下个主动回合的行动次数。

## 9. 换阵、借位与继承

若自己的 6 个卡槽都有卡，主动回合可以把行动用于换阵：

- 选择一个属性牌组抽 1 张。
- 若抽到一次性卡，可在对应属性槽发动，发动后弃置。
- 若该槽已有永续卡，一次性卡可借位发动，永续卡保留。
- 若抽到永续卡，可替换对应槽已有卡。
- 若新永续替换旧永续，新永续继承旧永续当前灵力，且不能超过新永续最大灵力。

中台换阵：

- 选择任意属性牌组抽 1 张。
- 可以放入中台或借中台发动。
- 中台永续替换时同样继承灵力。
- 若抽到阵眼卡，只能放入中台。
- 若中台已有阵眼卡，玩家可以主动更换；旧阵眼进入弃牌区，新阵眼立即生效。
- 若中台已有普通卡，阵眼卡可以替换该普通卡；被替换的普通卡进入弃牌区。
- 阵眼不能借位发动，它必须占据中台才生效。

## 10. 灵石与灵力

每名玩家在自己回合开始获得 1 枚灵石。

补灵：

- 每回合可以支付 1 枚灵石，为己方任意卡槽补充 1 点额外灵力。
- 每槽最多补充 2 点额外灵力。
- 额外灵力公开放置，即使卡牌盖伏，额外灵力数量也公开。
- 补灵后，灵石不再作为单独资源保留，而是直接成为该卡本次存在期间的灵力。

灵力结算：

- 攻击卡发动时，攻击灵力 = 基础灵力 + 额外灵力。
- 防御卡触发时，防御灵力 = 基础灵力 + 额外灵力。
- 反击卡触发时，反击灵力 = 基础灵力 + 额外灵力。
- 治疗卡发动时，治疗灵力 = 基础灵力 + 额外灵力。
- 销毁卡发动时，销毁灵力 = 基础灵力 + 额外灵力。
- 永续卡被攻击或销毁时，扣除对应灵力；灵力归 0 则破坏。

防守位触发：

- 防守位卡可以表侧或里侧放置。
- 防守位被攻击时，玩家先决定是否发动防守效果。
- 若发动，防守卡翻开并结算防御、反击、治疗或销毁等防守效果；一次性防守卡发动后弃置。
- 若不发动，攻击方灵力直接与该卡当前灵力相减；灵力归 0 则破坏，未归 0 则留场。
- 可以为里侧防守卡补灵，额外灵力公开。
- 当防守卡触发时，额外灵力自动计入防御、反击或治疗结算。

中台防守反击：

- 中台可以放置防御卡或带有防守反击效果的卡。
- 当对方攻击己方五行槽时，若中台卡属性与被攻击槽属性相同，或卡牌效果写明可以保护该槽，中台卡可以率先触发。
- 中台防守卡率先触发后，若攻击被完全抵消，被攻击槽不再触发防御卡。
- 若攻击仍有剩余伤害或后续效果，被攻击槽仍可继续触发自己的防御、永续被动和联协。
- 中台一次性防守卡触发后弃置，中台变空。
- 中台阵眼可以拥有防守反击效果；阵眼触发后不离场，除非效果特别写明。

## 11. 联协

当己方一个五行槽在被动回合被攻击，且该槽的卡牌触发防御或被动效果时，可按相生关系触发连续联协。联协是五行阵的基础规则，不要求卡牌写有“联协效果”才能发动。

联协方向：

```text
木槽触发 -> 火槽可联协
火槽触发 -> 土槽可联协
土槽触发 -> 金槽可联协
金槽触发 -> 水槽可联协
水槽触发 -> 木槽可联协
```

规则：

- 若被攻击的对应卡槽为空，对方可以直接攻击该空槽；空槽不能触发防御、被动效果或联协。
- 每一步联协都需要询问玩家是否发动。
- 若玩家选择发动，则该槽卡牌按自身类型结算一次基础效果。
- 攻击卡不能参与联协；若相生链的下一个槽是攻击卡，本次联协链停止。
- 发动后继续检查下一个相生槽，可以一直沿相生方向联协。
- 同一次连锁中，同一个五行槽最多发动 1 次，避免无限循环。
- 联协卡若为一次性卡，发动后弃置。
- 联协卡若为永续卡，触发一次基础效果，不离场。
- 联协卡所在槽的额外灵力会自动计入本次联协效果。
- 中台默认不参与基础联协，但部分卡牌可以让中台参与联协。

## 12. 阵眼压制

当双方中台都存在阵眼卡时，比较双方阵眼属性。阵眼之间不互相造成伤害，但会产生道统压制。

阵眼压制有两种来源：

1. 相克压制：己方阵眼属性克制对方阵眼属性。
2. 吸纳压制：对方阵眼属性生出己方阵眼属性。

例如：

- 火阵眼压制金阵眼，因为火克金。
- 火阵眼也压制木阵眼，因为木生火，火可以吸纳木势。

压制效果：

- 若己方阵眼压制对方阵眼，己方每回合首次发动的卡牌灵力效果翻倍。
- 可翻倍的灵力包括攻击、防御、反击、治疗、补灵、销毁等基础灵力。
- 灵石提供的额外灵力不翻倍。
- 永续卡和阵眼卡的准备阶段自动效果不翻倍，除非卡牌特别写明。
- 若双方阵眼互相形成压制，双方都获得压制效果。
- 若双方阵眼同属性，双方都不获得压制效果。

示例：

- 己方火阵眼，对方木阵眼。己方本回合首次发动火攻击卡，基础灵力 2 翻倍为 4；若本槽已有 1 点额外灵力，最终结算为 5。
- 己方火阵眼，对方木阵眼。己方首次发动治疗卡，基础灵力 2 翻倍为 4，即使治疗卡不是火属性也可以翻倍。

进阶测试规则：

- 如果 0.1 版压制不够有存在感，可以改为“每回合首次同属性卡牌效果翻倍，且首次非同属性卡牌效果 +1”。
- 如果压制过强，可以改为“每回合首次灵力效果 +1”，不再翻倍。

## 13. 胜利条件

玩家满足任意条件即失败：

- 生命值降到 0。
- 五个五行槽中有 3 个被对手压阵。
- 五个属性牌组全部抽空，且无法补阵。

单个属性牌组抽空：

- 该属性进入枯竭。
- 该属性槽为空时无法补阵。
- 枯竭不会立刻失败，但会增加被压阵风险。

## 14. 平衡基准

第一轮测试建议遵循：

- 普通攻击灵力：1 到 2。
- 高灵力攻击：3，但需要条件或副作用。
- 普通防御灵力：1 到 2。
- 普通治疗灵力：1 到 2。
- 永续灵力：3 到 5。
- 永续每回合收益：通常为 1 点灵力。
- 销毁卡对永续灵力消耗：2 到 3。
- 克制攻击灵力 x2。
- 补灵最多让单张卡获得 2 点额外灵力。
- 阵眼每回合收益应低于同费用永续，通常是改写规则、提供小幅调度、或每回合限 1 次触发。
- 阵眼不能直接提供稳定高额伤害或高额治疗，除非有弃牌、主动行动或阶段限制。
- 阵眼压制的翻倍应只影响每回合首次卡牌灵力效果，避免整回合爆发失控。

- 平均每局回合数。
- 永续卡平均存活回合。
- 阵眼平均存活回合，以及是否必须优先带销毁卡处理阵眼。
- 灵石是否经常堆满不用。
- 压阵胜利是否太容易或太难。
- 哪个属性最常被抽空。
- 主动回合只动一槽是否过于憋屈。
- 被动联协是否太强。
- 中台攻击任意槽是否过强。
- 中台防守率先触发是否让进攻太难。

如果游戏偏慢：

- 玩家生命从 24 降到 20。
- 永续灵力整体 -1。
- 克制攻击从 x2 改为 x3。

如果游戏偏快：

- 玩家生命从 24 提到 28。
- 压阵胜利从 3 槽改为 4 槽。
- 每槽额外灵力上限从 2 改为 1，降低爆发。

如果永续卡大战明显：

- 销毁卡基础灵力整体 +1。
- 永续触发效果需要补灵。
- 永续被克制攻击时，攻击灵力倍率从 x2 改为 x3。

如果阵眼过强：

- 每张阵眼增加“每回合限 1 次”。
- 允许销毁卡在中台发动时直接销毁阵眼。
- 玩家主动更换阵眼时，本回合不能再使用中台卡效果。

如果防御太强：

- 每次被动触发仍只允许 1 次联协。
- 防御卡不能同时增强防御和反击，只能二选一。
- 防御卡通过补灵增强时每次最多获得 1 点额外灵力。
